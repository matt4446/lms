# Quickstart: Manage Courses

## Creating a Course
1. Navigate to **Instructor Dashboard > Courses**.
2. Click **Create Course**.
3. Enter a **Title** and **Description**.
4. You will be redirected to the **Course Editor** (Draft mode).

## Publishing
1. In the Course Editor, add your content.
2. Click **Publish** in the top right corner.
3. The course is now visible to learners (if within Start/End dates).

## Creating a New Version
1. Open a published course.
2. Click **Edit** or **Create New Draft**.
3. A new version (e.g., v2) is created. The published version (v1) remains active.
4. Make changes to v2.
5. Click **Publish** to switch learners to v2.

## Exporting
1. Go to **Course Settings > Export**.
2. Click **Export Current Version**.
3. Wait for the process to complete.
4. Click **Download** to save the ZIP file.

## Importing
1. Go to **Instructor Dashboard > Courses**.
2. Click **Import Course**.
3. Select a `.zip` file (up to 2GB).
4. Wait for the upload and validation.
5. Choose **Import as New Course** or **Overwrite Existing**.
