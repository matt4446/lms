# Specification - Internal LMS Core

## Functional Requirements
1. **Learner Registration & Enrollment**: Users can sign up, view courses, and enroll.
2. **AI-Assisted Course Creation**: Course Owners can generate course structures using AI.
3. **Structured Learning & Gating**: Learners can navigate course content and are blocked by prerequisites.
4. **Exams & Results**: Assessments and grading.
